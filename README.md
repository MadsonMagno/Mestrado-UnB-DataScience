Este repositório centraliza minhas pesquisas acadêmicas, implementações e estudos realizados durante o Mestrado na **Universidade de Brasília (UnB)**. Minha pesquisa está posicionada na intersecção entre **Machine Learning** e **Cibersegurança**.

## 📑 Foco da Pesquisa: Priorização de Vulnerabilidades Baseada em Risco

Minha dissertação explora o uso de algoritmos de aprendizado de máquina para resolver o problema de "sobrecarga de vulnerabilidades" em grandes organizações. Ao correlacionar dados de scans internos (CWE) com inteligência de ameaças externa (EPSS), desenvolvi modelos preditivos para identificar quais vulnerabilidades representam o maior risco real para a infraestrutura.

### Principais Projetos Acadêmicos & Notebooks
- **[PPEEExplorarDataset_.ipynb](PPEEExplorarDataset_.ipynb):** Análise Exploratória de Dados (EDA) de grandes datasets de vulnerabilidades.
- **[Arvores_de_Decisao][Classificação].ipynb:** Implementação de algoritmos de Árvore de Decisão para classificação de risco.
- **[UESB]_HAR_KNN_.ipynb:** Pesquisa de Reconhecimento de Atividades Humanas usando K-Nearest Neighbors.
- **Motor de Scoring de Vulnerabilidades:** Lógica central para o motor de priorização baseado em risco.
