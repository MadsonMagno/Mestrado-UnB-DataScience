## 📑 Artigo Publicado

O principal resultado da minha pesquisa foi o artigo **"Uma análise comparativa de algoritmos de clusterização para a definição de perfis de risco de vulnerabilidades"**, publicado na *Revista Delos (v.19, n.79)*. Este trabalho foi desenvolvido em conjunto com o meu orientador, Prof. Dr. João Souza Neto.

🔗 **Link para o artigo:** [https://doi.org/10.55905/rdelosv19.n79-066](https://doi.org/10.55905/rdelosv19.n79-066)

### Resumo da Pesquisa
O trabalho propõe uma abordagem orientada a dados para categorizar riscos utilizando modelos de aprendizado não-supervisionado. Utilizando um dataset enriquecido com a probabilidade de exploração (EPSS) e impacto contextual, foram avaliados o desempenho dos algoritmos **K-Means**, **Clusterização Hierárquica Aglomerativa** e **DBSCAN**.

Os resultados destacam que o K-Means (k=5) ofereceu o melhor equilíbrio para a segmentação em larga escala, resultando em uma **redução de 69% no esforço de remediação** comparado aos métodos tradicionais. O DBSCAN provou ser crucial para a detecção de anomalias de alto risco (*outliers*).
